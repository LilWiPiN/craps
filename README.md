# CRAPS
Craps es un clásico juego de casino. Esta página es una versión adaptada del juego para navegadores.

## Comenzando
Para ejecutar el juego debes de entrar a la página https://lilwipin.github.io/craps/, no tienes que descargar ningún software ni nada por el estilo.

## Construido con
Esta página web fue realizada en el software de editor de texto **Sublime text 3**.
Utilizando los lenguajes de programación:
HTML5
CSS3
JavaScript
